# Parasat Product Astana Shop

Frontend-проект интернет-магазина для компании Parasat Product Astana.

## Стек

- React
- TypeScript
- Tailwind CSS
- React Router
- Vite

## Что реализовано

- главная страница;
- каталог товаров;
- поиск и фильтр по категориям;
- страница товара;
- корзина с изменением количества;
- оформление заявки без backend;
- страницы о компании и контактов;
- вход в админ-панель как визуальный макет;
- админ-панель;
- управление товарами как подготовленный интерфейс;
- управление заявками с временными данными.

## Запуск

```bash
npm install
npm run dev
```

Данные товаров находятся в `src/data/products.ts`, временные заявки - в
`src/data/orders.ts`.
