# Техническое задание проекта

## 1. Название проекта

**Интернет-магазин Parasat Product Astana**

## 2. Основание для разработки

Проект разрабатывается в рамках производственной практики студента 3 курса.

Компания **Parasat Product Astana** занимается продажей профессиональной химии и расходных материалов для гостиниц, ресторанов и кейтеринговых компаний. Также компания работает в сфере обеспечения HoReCa-сегмента оборудованием, сырьем, материалами, техническим сопровождением и автоматизацией бизнес-процессов 

## 3. Цель проекта

Цель проекта — разработать полноценный веб-сайт интернет-магазина для компании **Parasat Product Astana**, который позволит клиентам просматривать товары, добавлять их в корзину и отправлять заявку на заказ.

## 4. Задачи проекта

В ходе разработки необходимо выполнить следующие задачи:

1. Проанализировать деятельность компании Parasat Product Astana.
2. Определить основные категории товаров.
3. Разработать структуру сайта.
4. Создать пользовательский интерфейс интернет-магазина.
5. Реализовать каталог товаров.
6. Реализовать поиск и фильтрацию товаров.
7. Реализовать страницу подробного просмотра товара.
8. Реализовать корзину.
9. Реализовать форму оформления заявки.
10. Реализовать административную панель.
11. Реализовать добавление, редактирование и удаление товаров.
12. Реализовать просмотр заявок клиентов.
13. Подключить базу данных.
14. Проверить корректность работы сайта.
15. Подготовить скриншоты и описание проекта для отчета.

## 5. Целевая аудитория

Сайт предназначен для следующих пользователей:

### Клиенты

Клиентами являются:

* гостиницы;
* рестораны;
* кафе;
* кейтеринговые компании;
* клининговые организации;
* предприятия, которым необходима профессиональная химия и расходные материалы.

### Администратор

Администратор управляет товарами, категориями и заявками клиентов.

## 6. Основные разделы сайта

Сайт должен включать следующие страницы:

```text
/                  Главная страница
/catalog           Каталог товаров
/product/:id       Страница товара
/cart              Корзина
/checkout          Оформление заявки
/about             О компании
/contacts          Контакты
/admin/login       Вход администратора
/admin/dashboard   Панель администратора
/admin/products    Управление товарами
/admin/orders      Управление заявками
```

## 7. Категории товаров

В интернет-магазине будут представлены следующие категории:

1. Кухонная химия.
2. Средства для уборки.
3. Индустриальная химия.
4. Дезинфицирующие средства.
5. Прачечные средства.
6. Средства личной гигиены.
7. Бумажная продукция.
8. Одноразовая посуда.
9. Упаковочные материалы.
10. Средства индивидуальной защиты.

## 8. Функциональные требования

### Для клиента

Клиент должен иметь возможность:

1. Просматривать главную страницу сайта.
2. Читать информацию о компании.
3. Просматривать каталог товаров.
4. Фильтровать товары по категориям.
5. Искать товары по названию.
6. Открывать подробную страницу товара.
7. Добавлять товары в корзину.
8. Изменять количество товаров в корзине.
9. Удалять товары из корзины.
10. Оформлять заявку на заказ.
11. Указывать имя, телефон, город, компанию и комментарий.
12. Отправлять заявку администратору.

### Для администратора

Администратор должен иметь возможность:

1. Авторизоваться в системе.
2. Просматривать список товаров.
3. Добавлять новые товары.
4. Редактировать товары.
5. Удалять товары.
6. Просматривать заявки клиентов.
7. Менять статус заявки.
8. Просматривать основную статистику.

## 9. Статусы заявки

Для заявок будут использоваться следующие статусы:

```text
Новая
В обработке
Выполнена
Отменена
```

## 10. Данные товара

Каждый товар должен содержать:

```text
id
название
категория
описание
назначение
цена
изображение
наличие
дата добавления
```

Пример товара:

```json
{
  "id": 1,
  "name": "Briller",
  "category": "Кухонная химия",
  "description": "Средство для удаления накипи",
  "price": "Уточнить цену",
  "stock": true,
  "image": "/products/briller.jpg"
}
```

## 11. Данные заявки

Каждая заявка должна содержать:

```text
id
имя клиента
телефон
город
название компании
комментарий
список товаров
статус заявки
дата создания
```

Пример заявки:

```json
{
  "id": 1,
  "customerName": "Алия",
  "phone": "+7 777 777 77 77",
  "city": "Астана",
  "companyName": "Ресторан Example",
  "comment": "Нужна консультация по объему",
  "status": "Новая",
  "items": [
    {
      "productName": "Briller",
      "quantity": 2
    }
  ]
}
```

## 12. Нефункциональные требования

Сайт должен быть:

1. Адаптивным для компьютеров, планшетов и телефонов.
2. Удобным для пользователя.
3. Быстрым при загрузке страниц.
4. Безопасным для административной части.
5. Понятным для дальнейшего сопровождения.
6. Подходящим для демонстрации в рамках производственной практики.

## 13. Технологический стек

Для разработки проекта используется следующий стек:

```text
Frontend: React + TypeScript
Стилизация: Tailwind CSS
Маршрутизация: React Router
Backend/Database: Firebase
Авторизация администратора: Firebase Authentication
База данных: Firestore
Хранение изображений: Firebase Storage
```

## 14. Ожидаемый результат

В результате практики должен быть разработан веб-сайт интернет-магазина, который включает:

1. Главную страницу.
2. Каталог товаров.
3. Страницу товара.
4. Корзину.
5. Форму заявки.
6. Административную панель.
7. Базу данных товаров и заявок.
8. Адаптивный дизайн.
9. Готовые скриншоты для отчета.
